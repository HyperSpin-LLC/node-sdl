### `sdl.joystick`

#### `sdl.joystick.devices`

* `<object>[]`

This property holds a list of all the detected joystick devices.

#### `sdl.joystick.open(device[, options])`

* `device: <object>` An object from [`sdl.joystick.devices`](#sdljoystickdevices) that should be opened.
* Returns: `<Joystick>` a new [`Joystick`](#class-joystick) object.

Initializes a joystick device so it can start generating events.
